# House Price Prediction using Machine Learning

## Project Overview
This project predicts house prices using a Machine Learning model based on housing features such as bedrooms, bathrooms, square footage, and location.

The goal of this project is to build a predictive model that estimates house prices using historical housing data.

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## Machine Learning Model

The model used in this project is **Linear Regression**, which predicts house prices based on multiple housing features.

---

## Project Workflow

1. Import required Python libraries
2. Load the housing dataset
3. Perform data cleaning
4. Handle missing values and duplicates
5. Select important features
6. Split data into training and testing sets
7. Train the Linear Regression model
8. Predict house prices
9. Evaluate model performance

---

## Model Evaluation

The model performance is evaluated using:

- **Mean Absolute Error (MAE)**
- **R² Score**

These metrics help measure the accuracy of the predictions.

---

## Data Visualization

A scatter plot is used to compare:

- Actual house prices
- Predicted house prices

This helps visualize the performance of the model.
